# Freightos Assignment

This is the solution for the Freightos coding assignment on frontend React

This code base uses

- React (with Vite)
- Tailwind (for CSS)
- Shadcn/ui (UI lib)
- TanStack Query (for efficient fetch calls)

To run the code pls run the follow command after cloning the repo

```shell
  npm install
  npm run dev
```
